DialogueSystem.js

A JavaScript plugin that adds a div based RPG style dialogue system.


Cool Features

• Includes everthing needed for making dialogues
• Configurable to match certain games
• Animation system
•
• Uses game based update loop
• Managed text wrapping   (w     o    w)
• Extensible
• Literally just one file

Installation

To install the online version of DialogueSystem.js to your project, add this line of code to your page. 

HTML<script src="https://calmbubbles.github.io/js-plugins/DialogueSystem.js" type="text/javascript"></script>

Or you can just   Download it to use offline.
